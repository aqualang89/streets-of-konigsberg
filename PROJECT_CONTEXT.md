# PROJECT_CONTEXT.md

## Проект
Название: История улиц Калининграда
Тип: Контентный сайт
Цель: Рассказать историю улиц Калининграда/Кёнигсберга через видео (Рутуб) и текстовый контент с атмосферным историческим дизайном

## Стек
- Runtime: Browser (Vanilla JS)
- Framework: нет (чистый HTML + CSS + JS)
- Database: нет (статический контент)
- Auth: нет
- Deployment: Vercel (домен: streets-of-konigsberg.vercel.app)

## Дизайн-концепция
- Исторический стиль старого Кёнигсберга
- Цветовая палитра: сепия, тёплые коричневые, пергамент, тёмное дерево
- Готические/антикварные шрифты (Google Fonts)
- CSS-текстуры: старая бумага, пергамент, виньетки — без картинок
- Scroll-анимации (появление элементов, параллакс)
- SVG-орнаменты и декор в прусском стиле
- Плавные переходы и hover-эффекты

## Контент
- Видео с Рутуба (embed)
- Историческая справка по каждой улице
- Старое и новое название улицы
- Язык: русский (в будущем — переключение RU/EN)

## Архитектура
Многостраничный статический сайт. Главная + страницы отдельных улиц.

## Структура папок
```
streets-of-konigsberg/
├── index.html              # Главная страница
├── streets/
│   └── example-street.html # Страница улицы (шаблон)
├── css/
│   ├── style.css           # Основные стили
│   ├── animations.css      # Анимации и эффекты
│   └── fonts.css           # Подключение шрифтов
├── js/
│   ├── main.js             # Основная логика
│   ├── animations.js       # Scroll-анимации
│   └── i18n.js             # Переключение языка (будущее)
├── assets/
│   ├── svg/                # SVG-орнаменты и декор
│   └── images/             # Фото (добавит Макс)
├── PROJECT_CONTEXT.md
└── PROJECT_STATE.md
```

## Модули
- main: навигация, общая логика
- animations: scroll-triggered анимации, параллакс, появление элементов
- i18n: мультиязычность RU/EN (будущее)

## Правила
- Никаких изменений архитектуры без согласования
- Код production-ready
- Функции модульные
- Новые зависимости только с явного одобрения
- Дизайн максимально атмосферный — CSS-first подход
- Фото добавляет Макс вручную

## Стандарты кода
- Язык: JavaScript (ES6+)
- Стиль: чистый, читаемый, с комментариями на русском
- Тесты: ручное тестирование перед деплоем
